# backup-drive
```
Requirements:
- Python 3.10.12 or higher
- Follow this instruction to get credentials.json file:
  - Change name to credentials_source.json for source drive
  - Change name to credentials_destination.json for destination drive
 ```
```
HOW TO USE:
- Run the following command to install the required packages:
    pip install -r requirements.txt
- Change the source and destination folder in the main.py file
- Run the following command to start the program:
    python main.py
- Error is log to error.log file 
```
